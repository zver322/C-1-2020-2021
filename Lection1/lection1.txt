1.2 Знакомство с омпилятором и первая программа.
vim test.cpp - использовать редактор vim, чтобы начать редактировать файл test.cpp
:q! - использовать, чтобы выйти из редактора. Это без созранения изменений
:wq - выйти с сохранением изменений
Чтобы начать печатать нажать клавишу I
Esc, чтобф выйти из режима 

Компиляция - надо создать исполняемый файл из кода прежде чем начать его выполнять.
Для компиляции файла надо использовать команду: g++ test.cpp

Для запуска исполняемого файла надо просто использовать команду: ./a.out

Можно использовать другой компилятор clang++
Чтобы скомпилировать файл надо использовать команду clang++ test.cpp
